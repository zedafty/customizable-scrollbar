What's done

  • Attach CSS customizable scrollbars to any HTML container
  • Display scrollbars at different positions and places
  • Hide scrollbars when container has no scrollable content
  • Control scrollbars buttons and track visibility
  • Based scrollbar behaviour upon Windows style

What's NOT done

  • Selection scroll by keyboard
  • Middle mouse button scroll compass
  • Browser smooth scroll ability Experimental
  • Nested scrollbars

Do NOT accept Global options

Override Default options
